Conceptos básicos de NPM

  • npm i -> instalar
  • npm start -> arrancar poryecto
  • npm i -g json-server -> instalar json-server en global
  • npm i --save-dev json-server -> instalar en el proyecto en el package.json
  • npm i --save-dev concurrently -> instalar una libreria para arrancar comandos automaticamente al arrancar la aplicacion
  • npm run dev -> arrancar el servidor http
  • En el VCS, hacemos Terminal -> Run Build Task... -> tsc: watch - tsconfig.json, lo cual ejecuta el siguiente comando: tsc -p c:\1713\router-intro\tsconfig.json --watch

En el package.json, en las librerias cuando aparece el simbolo ^, significa esa versión o mayor. Por ejemplo, ^1.6.9 significa, 1.6.9 o superior
